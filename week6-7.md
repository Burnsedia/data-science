# Weeks 6-7: Python Data Analysis

## Overview
Focus on pandas, EDA, and visualization. Build skills for real data work. Estimated time: 15-20 hours.

## Learning Outcomes
- Manipulate data with pandas.
- Perform exploratory data analysis (EDA).
- Create visualizations with matplotlib, seaborn, and Plotly.

## Lectures (8-10 hours)
- [freeCodeCamp – Data Analysis with Python (5 hrs)](https://www.youtube.com/watch?v=r-uOLxNrNk8)
- [Kaggle – Pandas Course](https://www.kaggle.com/learn/pandas)
- [Kaggle – Data Visualization Course](https://www.kaggle.com/learn/data-visualization)
- Modern: [Plotly Python Docs](https://plotly.com/python/) - For interactive charts.

## Interactive Projects (7-10 hours)
- Finish [freeCodeCamp Data Analysis with Python Certification](https://www.freecodecamp.org/learn/data-analysis-with-python/).
- EDA on Netflix dataset: Use pandas + seaborn/Plotly.
