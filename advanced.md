# Advanced Track: Neovim + CLI Tools

This optional track enhances productivity with modern CLI tools and Neovim. For power users.

## Neovim Setup
- Install: `brew install neovim` (Mac), `apt install neovim` (Ubuntu), `pacman -S neovim` (Arch).
- Basics: `i` insert, `:wq` save, `dd` delete line.
- Tutorial: [freeCodeCamp Vim (2 hrs)](https://www.youtube.com/watch?v=RZ4p-saaQkc).
- Config: Edit `~/.config/nvim/init.lua` for plugins.

## CLI Tools
- fzf: `brew/apt/pacman install fzf` - Fuzzy finder.
- ripgrep: `brew/apt/pacman install ripgrep` - Fast search (`rg` command).
- htop: `brew/apt/pacman install htop` - System monitor.
- bat: `brew/apt/pacman install bat` - Better `cat`.
- exa: `brew/apt/pacman install exa` - Better `ls`.
- tmux: `brew/apt/pacman install tmux` - Terminal multiplexer.

## Practice
- Replace commands: `cat → bat`, `ls → exa`.
- Use fzf for history: Bind to Ctrl+R.
- Search code: `rg "function"` in projects.
- Monitor: `htop` for processes.
- Multi-task: `tmux` for sessions.
