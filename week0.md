# Week 0: CS & Unix/Linux

## Overview
This week introduces computer science fundamentals and Unix/Linux command-line basics. You'll learn to navigate file systems, use shell tools, and understand core CS concepts. Estimated time: 10-15 hours.

## Learning Outcomes
- Understand basic CS principles (algorithms, data structures).
- Navigate Linux terminals confidently.
- Use essential shell commands for file management.

## Lectures (5-7 hours)
- [CS50's Introduction to Computer Science (Harvard)](https://cs50.harvard.edu/x/) - Full course; focus on weeks 1-2.
- [freeCodeCamp – Linux for Beginners (4 hrs)](https://www.youtube.com/watch?v=sWbUDq4S6Y8) - Practical Linux commands.
- [MIT Missing Semester – Shell Tools & Scripting](https://missing.csail.mit.edu/2020/shell-tools/) - Advanced shell usage.
- Bonus: [Khan Academy – Intro to Algorithms](https://www.khanacademy.org/computing/computer-science/algorithms) - 1-2 hours.

## Interactive Projects (3-5 hours)
- [OverTheWire – Bandit Wargame](https://overthewire.org/wargames/bandit/) - Hands-on Linux terminal practice (levels 1-10).
- Practice: Create directories, move files, use `ls`, `cd`, `grep`, `find`.

## Resources
- [Linux Command Line Basics](https://ubuntu.com/desktop/tutorial/command-line-for-beginners) - Ubuntu guide.
- [Shell Scripting Tutorial](https://www.shellscript.sh/) - For advanced users.
