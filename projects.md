# Project Directory

This directory catalogs hands-on projects by week, with detailed steps, code examples, and assessments. All projects integrate Python, R, PydanticAI, and Ollama.

## Project Assessment Rubric
- **Completeness (1-5)**: All requirements met?
- **Code Quality (1-5)**: Readable, efficient, uses best practices?
- **Innovation (1-5)**: Added features or improvements?
- **Submission**: GitHub repo with README; total score for pass: 10+.

## Week 0: Command Line File Audit Tool
- **Goal**: Python CLI to summarize files in a directory.
- **Steps**: Use `os` module; output CSV.
- **Code Example**:
  ```python
  import os
  # List files, count types
  ```
- **Stretch**: Add LLM summary with PydanticAI.
- **Assessment**: Submit repo; self-grade rubric.
