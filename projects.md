# Project Roadmap

This roadmap details hands-on projects for each week, integrating tools learned. All projects use Python, R, PydanticAI, and Ollama.

## Week 0: Command Line File Audit Tool
- **Goal**: Python CLI to summarize files in a directory.
- **Steps**: Use `os` module; output CSV.
- **Code Example**:
  ```python
  import os
  # List files, count types
  ```
- **Stretch**: Add LLM summary with PydanticAI.

## Week 1: Python Expense Tracker (CLI)
- **Goal**: Log expenses, categorize.
- **Steps**: CLI input, save to CSV with pandas.
- **Output**: Terminal charts with `rich`.

## Week 2: CSV Column Analyzer
- **Goal**: Analyze CSV columns (types, nulls, stats).
- **Steps**: Use pandas; LLM rephrase with PydanticAI.

## Week 3: Netflix Dataset EDA
- **Goal**: Explore dataset with pandas + viz.
- **Steps**: Load data; ask Ollama for plot suggestions.

## Week 4: Survey Summary Generator
- **Goal**: Calculate stats; LLM Markdown report.

## Week 5: R Data Explorer
- **Goal**: R report with dplyr + ggplot2.
- **Steps**: Knit to HTML.

## Week 6: Airbnb Dataset EDA Bot
- **Goal**: CLI for questions, powered by Ollama.

## Week 7: Dirty CSV Fixer
- **Goal**: LLM suggests fixes with PydanticAI schema.

## Week 8: CSV Chat Agent
- **Goal**: Natural language queries with ChromaDB.

## Week 9: PDF Report Analyzer
- **Goal**: Extract text with PyMuPDF; LLM summary.

## Week 10: Natural Language to SQL
- **Goal**: LLM generates SQL for queries.

## Weeks 11-12: Analyst Copilot
- **Goal**: Multi-modal assistant (PDF/CSV/SQL).
- **Steps**: Streamlit UI; host on GitHub.
