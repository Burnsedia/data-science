# Weeks 8-12: LLMs & AI Agents + Capstones

## Overview
Integrate LLMs with data tools. Learn SQL, build AI agents, and capstone projects. Estimated time: 25-35 hours.

## Learning Outcomes
- Use PydanticAI with Ollama for AI tasks.
- Query databases with SQL.
- Build end-to-end data + AI applications.

## Lectures (10-15 hours)
- [freeCodeCamp – LangChain for Beginners](https://www.youtube.com/watch?v=lG7Uxts9SXs) (concepts apply to PydanticAI).
- [DeepLearning.AI – Free LLM Courses](https://learn.deeplearning.ai/).
- [freeCodeCamp – SQL Full Course (4 hrs)](https://www.youtube.com/watch?v=HXV3zeQKqGY).
- [Khan Academy – Intro to SQL](https://www.khanacademy.org/computing/computer-programming/sql).

## Interactive Projects (15-20 hours)
- [freeCodeCamp – Relational Database Certification](https://www.freecodecamp.org/learn/relational-database/) - Projects: Celestial Bodies DB, World Cup DB, etc.
- [freeCodeCamp – Machine Learning with Python](https://www.freecodecamp.org/learn/machine-learning-with-python/) - Projects: Rock-Paper-Scissors AI, etc.
- Capstone: Analyst Copilot with PDF + CSV + SQL + LLM integration.
