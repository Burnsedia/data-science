# Week 1: Python Basics

## Overview
This week covers Python fundamentals: syntax, data types, loops, and conditionals. Estimated time: 6-10 hours.

## Learning Outcomes
- Write basic Python scripts.
- Understand variables, lists, and control flow.

1. **Coding Task**: Write a Python script that prints "Hello, Data Science!" and calculates the sum of numbers 1 to 10. Submit code via GitHub.
2. **Quiz**: What is the difference between a list and a tuple? (Answer in a comment in your script.)
3. **Reflection**: Describe one challenge you faced and how you solved it.
