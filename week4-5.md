# Weeks 4-5: Statistics & R

## Overview
Dive into statistics and R programming. Learn data visualization and basic stats. Estimated time: 15-20 hours.

## Learning Outcomes
- Understand mean, median, distributions, and hypothesis testing.
- Use R for data manipulation with Tidyverse.
- Create basic plots and reports.

## Lectures (8-12 hours)
- [Khan Academy – Statistics & Probability](https://www.khanacademy.org/math/statistics-probability)
- [StatQuest with Josh Starmer](https://www.youtube.com/user/joshstarmer) - YouTube playlists.
- [MIT OCW – Probability & Statistics](https://ocw.mit.edu/courses/res-6-012-introduction-to-probability-spring-2018/)
- [freeCodeCamp – R Programming for Data Science (2 hrs)](https://www.youtube.com/watch?v=_V8eKsto3Ug)
- [R for Data Science (Book)](https://r4ds.had.co.nz/) - Free online.

## Interactive Projects (7-8 hours)
- [freeCodeCamp – Data Analysis with Python Certification](https://www.freecodecamp.org/learn/data-analysis-with-python/)
  - Projects: Mean-Variance Calculator, Demographic Data Analyzer, Medical Data Visualizer, Time Series Visualizer, Sea Level Predictor.
- R Project: Use `dplyr` and `ggplot2` to analyze a dataset (e.g., from Kaggle).

## Resources

- [Tidyverse Cheat Sheet](https://rstudio.com/resources/cheatsheets/) - Quick reference.
